== Description ==

Nav Menu Query Meta Box (NMQMB) allows you to quickly add a post or a taxonomy to your menu by specifying its ID. NMQMB adds an additional meta box to your menu sidebar for your themes, from which you choose whether you want a insert a post or a taxonomy, and its ID.

Nav Menu Query Meta Box depends on WP Ajax Query Plugin.

== Installation ==

1. Install and activate <a href="http://blog.aizatto.com/wordpress-plugins/wp-ajax-query/">WP Ajax Query</a>
1. Upload the `wp-ajax-query` folder into your plugin directory.
1. Activate the plugin through the 'Plugins' menu in WordPress

== Why I Created It ==

I had to maintain a WordPress installation with over 2000+ articles and 800+ categories, and needed a quick way to insert a post or category into the menu.
